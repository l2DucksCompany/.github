# .github
L2 Ducks é um ecossistema completo para um servidor privado de Lineage 2 HighFive. Inclui o game server    (L2J), uma API de sincronização em tempo real, portal web com rankings, eventos, gestão de conta e   loja de Duck Coins, launcher desktop multiplataforma e bot Discord para a comunidade. Powered by   Firebase, React, Fastify e Electron.
